# 🐍 Snake Game (HTML One-File Version)

키보드 **화살표 키**로 조작하는 클래식 스네이크 게임입니다.  
HTML, CSS, JavaScript가 **한 파일에 통합된 형태**로 구성되어 있어 브라우저에서 바로 실행할 수 있습니다.

---

## 🎮 게임 특징

- HTML `<canvas>` 기반의 가벼운 스네이크 게임  
- **한 파일로 구성**되어 배포 및 실행이 매우 간단  
- 화살표 키로 조작  
- 먹이를 먹을 때마다 길이가 증가  
- 벽 충돌 및 자기 몸 충돌 시 게임 오버  
- 점수 표시 및 **게임 오버 화면 + 재시작 버튼** 제공
