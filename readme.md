🐍 Snake Game (HTML One-File Version)
키보드 화살표 키로 조작하는 클래식 스네이크 게임입니다.
HTML, CSS, JavaScript가 한 파일에 통합된 형태로 구성되어 있어, 브라우저에서 바로 실행할 수 있습니다.

🎮 게임 특징
HTML <canvas> 기반의 가벼운 스네이크 게임

한 파일로 구성되어 배포 및 실행이 매우 간단

화살표 키로 조작

먹이를 먹을 때마다 길이가 증가

벽 충돌 및 자기 몸 충돌 시 게임 오버

점수 표시 및 게임 오버 화면 + 재시작 버튼 제공

📂 파일 구성
이 프로젝트는 단 하나의 파일로 구성됩니다:

코드
index.html
HTML 내부에 CSS와 JS가 모두 포함되어 있어 별도의 파일이 필요 없습니다.

🚀 실행 방법
이 저장소를 다운로드하거나 클론합니다.

index.html 파일을 더블 클릭하여 브라우저에서 실행합니다.

화살표 키로 스네이크를 조작하며 게임을 즐기세요!

⌨️ 조작법
키	동작
⬆️ 위쪽 화살표	위로 이동
⬇️ 아래쪽 화살표	아래로 이동
⬅️ 왼쪽 화살표	왼쪽 이동
➡️ 오른쪽 화살표	오른쪽 이동
🧩 게임 화면
스네이크는 초록색으로 표시됩니다.

먹이는 빨간색으로 표시됩니다.

게임 오버 시 점수와 함께 재시작 버튼이 나타납니다.

📜 코드 구조
HTML: Canvas 및 UI 요소 구성

CSS: 게임 화면 스타일링

JavaScript:

스네이크 이동 로직

먹이 생성

충돌 감지

점수 관리

게임 오버 처리

🛠️ 수정 및 확장 아이디어
난이도 조절 (속도 변경)

벽 통과 모드 추가

스킨 변경

모바일 터치 조작 추가

최고 점수 저장 기능(LocalStorage)
